CVPR 2026 Findings

BeyondSingleObject: Learning 3D Relations with Large Language Models

Kohsuke Ide, Ryousuke Yamada, Yue Qiu, Xianzheng Ma, Yoshihiro Fukuhara, Hirokatsu Kataoka, Yutaka Satoh

Code Citation Data Weights

Abstract

BeyondSingleObject addresses multi-object 3D reasoning with MO3D, a PointLLM-based Multi-3DLLM architecture, and two application-driven mini-app benchmarks: Shape Mating and Change Captioning.

Contributions

MO3DInstruction data for positional, comparative, and holistic multi-object 3D QA.
Multi-3DLLMA Patch-Interaction Transformer that models inter-object and intra-object patch relations.
Mini-AppsShape Mating and Change Captioning benchmarks for geometric understanding.

Quick Start

git clone https://github.com/KohsukeIde/BeyondSingleObject.git
cd BeyondSingleObject
conda env create -f environment.yml
conda activate beyond-single-object
pip install -e .

# Prepare data and weights as described in the README, then run:
MODEL_PATH=checkpoints/multi-3dllm ANNO_PATH=data/mo3d/test.json DATA_PATH=data/point_clouds scripts/eval/infer.sh
